# 隱私權政策（Privacy Policy）

最後更新：2026-04-23

本隱私權政策適用於「TPE Airport Tool」（以下稱「本服務」）。本服務透過 LINE 官方帳號／LINE Bot 提供資訊查詢與回覆功能，包含（但不限於）桃園機場入境人數／整點人數預估資訊，以及延伸的車資與排班數量預估等。

## 1. 我們蒐集哪些資料

為了提供與維持本服務運作，本服務可能會處理下列資料：

- LINE 使用者識別資訊：如 **userId**（LINE 產生，用於辨識訊息來源與回覆對象）。
- 互動內容：你在 LINE 對話中輸入的文字內容（例如指令或查詢條件）。
- 基本技術資訊：例如請求時間、錯誤紀錄（log）等，用於除錯與提升穩定性。

本服務不主動要求你提供姓名、電話、身分證字號、付款資料、精準定位等敏感個人資料。若你自行在對話中輸入個資，視為你主動提供，本服務仍會依本政策處理與保護。

## 2. 蒐集與使用目的

我們僅在下列目的範圍內使用資料：

- 回覆你在 LINE 中的查詢與指令
- 產出機場人流、車資與排班等相關預估資訊（供你參考）
- 系統維運、故障排除、安全防護與服務品質改善

## 3. 資料保存期間

- 本服務**以即時回覆為主**，原則上不需長期保存對話內容。
- 為維運需要，系統可能保留有限期間的技術紀錄（如錯誤日誌）以利排錯。
- 若未來因功能需求需要保存更多資料，我們會更新本政策並明確告知保存項目與期間。

## 4. 資料分享與第三方

我們不會出售或出租你的個人資料。以下情況可能會接觸到資料：

- LINE 平台：你的訊息與回覆會經由 LINE 提供的系統傳遞。
- 依法令或主管機關要求：在法律要求下可能揭露必要資訊。
- 資安與維運：為防止濫用、攻擊或系統故障，可能使用必要的安全機制與紀錄。

## 5. 你的權利

你可以透過停止使用本服務、封鎖／刪除好友等方式終止互動。若你希望查詢、刪除或限制我們對你的資料處理，可依下方聯絡方式提出。

## 6. 安全措施

我們會採取合理的技術與管理措施（例如存取控制、最小權限、避免將密鑰公開等）以降低資料外洩風險。但網路傳輸無法保證 100% 安全，仍請理解相關風險。

## 7. 政策變更

我們可能因服務調整或法規變動更新本政策。更新後會以本頁面公告為準。
